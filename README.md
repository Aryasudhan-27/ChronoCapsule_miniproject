# ChronoCapsule_miniproject
ChronoCapsule is a web-based application that allows users to upload files, encrypt them, and lock them until a selected future release date. The system uses password-based AES encryption and stores encrypted data securely in the cloud.

Users can:
-Sign up and log in securely
-Upload a file
-Encrypt the file with a password
-Set a future release date
-Unlock and decrypt the file only after the release date
